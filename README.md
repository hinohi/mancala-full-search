# Mancala

## ゲームルール

### 基本ルール

TBA

### 追加ルール

#### 石取り

動かした後に最後に置いた場所にある石の数が1だった場合（動かす前に空だった場合が多い）に、向かい側の相手の石全てと自分の石を取りスコアに加える。

### 勝利条件

どちらかの石がなくなったら終了ゲーム終了。

#### スコア性

スコアと盤面に残っている分を合わせたものが最終スコア。
スコアが大きいほうが勝ち。

#### 早なくし

先に盤面の石を全てなくしたほうが勝ち。
